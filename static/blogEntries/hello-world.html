technical overview of the blog

I wrote a mini-framework for writing these blog entries using SvelteKit. Utilizing a static-site generator like Hugo or Jekyll would be less effort, but I wanted to have the flexibility to add JS components on my site while learning a frontend framework. That being said, I would recommend the previously mentioned static-site generators if you wanted to have a quick and easy blog.
